What is Saturn Launcher?

Saturn Launcher is a lightweight console launcher for Minecraft, written in Python. It allows you to quickly and easily launch Minecraft with support for mods, shaders and custom servers.

Perfect for players who prefer a minimalist approach and want full control over the game launch process.

How to Download and Run

1

Download the launcher

Go to the releases page and download the appropriate version for your OS

2

Create a folder

Create a dedicated folder (e.g., SaturnLauncher) and place the downloaded file there

3

Run the launcher

For Linux make the file executable: chmod +x saturn_linux, then run ./saturn_linux

4

Play!

Select a Minecraft version, enter your username and enjoy the game
